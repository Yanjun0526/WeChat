## Writing Emphasis

The final report should place most emphasis on:

- methodology
- implementation details
- empirical results
- discussion of results
- conclusions drawn from the analysis

The literature review should support the research gap and methodological choices, but it should not dominate the final report. The empirical chapters should clearly explain what was implemented, how the models were built, what the results show, how the results should be interpreted, and what limitations remain.

## Front Matter

Expected front matter sequence:

1. Title page
2. Abstract
3. Declaration
4. Acknowledgements, if needed
5. Table of contents
6. List of figures
7. List of tables

The acknowledgements page is optional. If no acknowledgements are needed, remove it and renumber subsequent front-matter pages.

The title page should include the university/department, programme, dissertation module/final report label, thesis title, student name and student number if required, supervisor name, and date of submission.

The declaration page should state that the dissertation is the student's own work and has not been previously submitted for another degree, diploma, or qualification. It should include a signature line, printed name, and date.

## Abstract

- Target length: about 400-500 words.
- The abstract should be an abbreviated and accurate representation of the report.
- It should not be a general introduction.
- It should include major results, conclusions, or recommendations.
- It should include supporting details of method, scope, or purpose only as needed.
- The abstract page should not display a page number.

## Pagination And Layout

- Front matter after the abstract should use roman numerals.
- Chapter 1 should restart with Arabic page number 1.
- Use 25 mm page margins on all sides unless the final university template or supervisor instruction requires a different setting.
- Use 12 pt body text. Chapter headings may use the template's 12-14 pt style.
- Use clear paragraph indentation for new paragraphs.
- Follow the report template's line-spacing convention. If no later template overrides it, use double or sufficiently spacious line spacing for readability.
- Keep subsection depth readable. Avoid excessive nesting such as `3.6.1.1.1`.

## Figures, Tables, And Equations

- Number figures by chapter, such as `Figure 1.1`.
- Number tables by chapter, such as `Table 3.1`.
- Number equations by chapter, such as `(3.14)`.
- Place figure captions below figures.
- Place table titles above tables.
- Treat figure and table captions as titles. Avoid a final full stop unless the caption is a complete sentence or the chosen style requires it.
- Add legends or notes below tables when symbols, arrows, abbreviations, or special notation require explanation.

## Citation And Bibliography

- Use author-year citation style consistently unless the user changes the citation style.
- Use family names in in-text citations; do not confuse family names with given names.
- Do not invent sources or references.
- Place the bibliography after the conclusion and before appendices.

## Appendices

- Appendices should follow the bibliography.
- Use appendices for derivations, extended robustness checks, supplementary tables, code details, or material that supports the thesis but would interrupt the main argument.
- Appendix numbering should remain clear and linked to relevant chapters or analyses.
- A publications page may be added after the appendices only if relevant; otherwise omit it.

## Conclusion Requirements

The conclusion should cover:

- main achievements and findings
- contributions, innovations, or practical implications
- limitations
- directions for future research

For this thesis, conclusions should be drawn from the Stage 1-4 analysis outputs and should not make causal claims unless a causal identification strategy is explicitly added.
